Întrebări Frecvente

Aveți Întrebări?

Găsiți răspunsuri la cele mai frecvente întrebări despre serviciul nostru de chat online.

Ce tip de probleme pot raporta prin chat?

Serviciul de chat online este destinat raportării problemelor non-urgente, cum ar fi:

  • Activități suspecte care nu prezintă pericol iminent
  • Probleme de parcare sau circulație
  • Tulburarea ordinii publice
  • Probleme legate de mediu (deșeuri, poluare fonică)
  • Întrebări generale despre reglementări

Important: Pentru orice situație care reprezintă un pericol imediat pentru viață sau proprietate, vă rugăm să sunați la numărul de urgență 112.

Cât de sigură este comunicarea prin chat?

Siguranța și confidențialitatea sunt prioritățile noastre. Sistemul nostru de chat implementează:

  • Criptare TLS pentru toate comunicațiile
  • Stocarea securizată a datelor conform GDPR
  • Acces restricționat la informații doar pentru personalul autorizat
  • Opțiune de anonimitate pentru raportări

Toate conversațiile sunt stocate în sisteme securizate și pot fi accesate ulterior doar cu autorizare corespunzătoare pentru investigații oficiale.

Când este disponibil serviciul de chat?

Serviciul de chat online al Poliției Locale Slobozia este disponibil 24/7. Agenții noștri sunt pregătiți să răspundă în orice moment, ziua sau noaptea.

În perioadele de volum ridicat, timpul de așteptare poate varia, dar sistemul nostru vă va informa despre timpul estimat de așteptare și vă va oferi opțiunea de a lăsa un mesaj care va fi procesat cât mai curând posibil.

Pot rămâne anonim când folosesc chat-ul?

Da, puteți utiliza serviciul nostru de chat fără a vă dezvălui identitatea pentru raportări care nu necesită urmărire. Sistemul nu înregistrează automat date personale sau IP-uri.

Cu toate acestea, pentru cazurile care necesită investigații suplimentare sau urmărire, este util să furnizați informații de contact. Puteți opta să primiți transcriptul conversației și actualizări prin email fără a dezvălui alte date personale.

Pot atașa fișiere și fotografii în chat?

Da, sistemul nostru de chat suportă atașarea de:

  • Fotografii (JPG, PNG) - până la 10MB
  • Videoclipuri scurte (MP4) - până la 30MB
  • Documente (PDF) - până la 5MB

De asemenea, puteți opta să partajați locația dvs. direct din browser, ceea ce poate ajuta agenții să identifice exact zona despre care raportați.

Confidențialitate

Datele Tale sunt Protejate

La Poliția Locală Slobozia, ne angajăm să protejăm confidențialitatea și securitatea informațiilor dumneavoastră.

shield

Politica noastră de confidențialitate

Toate conversațiile prin serviciul nostru de chat sunt criptate end-to-end și stocate în siguranță pe servere securizate. Datele personale sunt procesate conform reglementărilor GDPR și legislației naționale privind protecția datelor.

Informațiile furnizate în timpul unei sesiuni de chat sunt utilizate exclusiv pentru:

  • Rezolvarea problemei raportate
  • Îmbunătățirea serviciilor noastre
  • Analiză statistică (date anonimizate)
  • Conformarea cu obligațiile legale

Conversațiile pot fi înregistrate și utilizate pentru antrenarea personalului, dar numai după anonimizarea datelor personale. Atașamentele și fotografiile sunt stocate separat, cu acces restricționat doar pentru personalul autorizat.

Orice informație furnizată către Poliția Locală Slobozia prin intermediul chat-ului este tratată cu cel mai înalt nivel de confidențialitate și securitate, urmând aceleași protocoale stricte ca și cele utilizate pentru alte canale oficiale de comunicare.

Arhitectură Tehnică

Tehnologii Utilizate

Sistemul nostru de chat online este construit pe o infrastructură modernă, securizată și scalabilă.

Arhitectură Sistem

computer Front-End

  • • Widget Chat: React + Socket.IO pentru comunicare în timp real
  • • Dashboard Ofițer: React + Material UI pentru interfața ofițerilor
  • • Responsive Design: Compatibil cu toate dispozitivele, inclusiv mobile

dns Back-End

  • • Server Chat: Node.js + Express + WebSocket (ws)
  • • Bază de Date: PostgreSQL cu JSONB pentru mesaje
  • • Stocare Media: MinIO (S3-compatibil) pentru atașamente

security Securitate

  • • Autentificare: Keycloak (OpenID Connect, OAuth2) și LDAP
  • • Criptare: TLS pentru toate comunicațiile, criptare la stocare
  • • Protecție: WAF, rate-limiting, fail2ban pentru prevenirea atacurilor

settings_applications Operațiuni

  • • Monitorizare: Prometheus + Grafana pentru metrici sistem
  • • Logging: ELK Stack pentru centralizare loguri și audit
  • • Deployment: Docker Compose / Kubernetes (k3s) pentru orchestrare
